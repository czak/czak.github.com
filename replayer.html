Replayer

Replayer is my first Mac OS X application, available now in the Mac App Store.

Replayer is an audio player for any practicing musician. Slow down playback of an audio file, while retaining its pitch. This way you can practice harder passages by slowing down original recordings and playing along.

I've been using the app myself to learn a number of guitar solos. You can see me struggling with my guitar on my YouTube channel :)

I wanted the app to be as easy to use as possible so that it would not interfere while playing the instrument. All functions can be accessed by one-hand keyboard shortcuts.
